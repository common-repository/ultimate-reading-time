Ultimate solution for displaying reading time on your posts and pages.

== Description ==

The Ultimate Reading Time plugin provides an easy way to display the estimated reading time for your posts and pages. This plugin allows you to customize the reading time display, including the text, color, font style, and position. Enhance user experience by giving readers an idea of how long it will take to read your content.

=== Features ===

* Display reading time on posts and pages.
* Customize reading time text prefix and postfix.
* Set custom words per minute for reading time calculation.
* Choose from multiple font styles and weights.
* Position reading time above or below content or title.
* Shortcode support for displaying reading time anywhere.
* Customizable colors for the reading time display.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ultimate-reading-time` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to "Reading Time" settings under the "Settings" menu to configure the plugin.

== Frequently Asked Questions ==

= How do I customize the reading time display? =

Navigate to the "Reading Time" settings page under the "Settings" menu. From there, you can customize various aspects of the reading time display, including text, color, font style, and position.

= Can I display the reading time using a shortcode? =

Yes, you can use the `[urtbenz_reading_time]` shortcode to display the reading time in your posts or pages. You can also specify a post ID like this: `[urtbenz_reading_time id=123]`.

== Usage ==

= Shortcodes =

* `[urtbenz_reading_time]` – Display the reading time for the current post or page.
* `[urtbenz_reading_time id=123]` – Display the reading time for a specific post or page with ID 123.

= Settings =

Navigate to "Reading Time" settings under the "Settings" menu to access the following options:

* **Display Options**: Choose where to display the reading time (posts, pages).
* **Color Settings**: Set a custom color for the reading time text.
* **Font Settings**: Customize the font style, size, and weight.
* **Position Settings**: Choose the alignment and position of the reading time display.
* **WPM & Label Settings**: Set custom words per minute and customize the reading time text prefix and postfix.
